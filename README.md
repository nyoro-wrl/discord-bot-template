# Discord Bot Template
TypeScriptを使用した、簡易的なDiscord Botのテンプレート。

## セットアップ
```bash
# パッケージのインストール
npm install
```

## 実行
```bash
# 開発環境
npm run dev
```
```bash
# 本番環境
npm start
```

## コマンドの作成
スラッシュコマンドはsrc/commands/chatに、ユーザーコンテキストメニューはsrc/commands/userに、メッセージコンテキストメニューはsrc/commands/messageにそれぞれ作成する。
